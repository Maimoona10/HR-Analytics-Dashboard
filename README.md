# HR Analytics Dashboard

## Project Overview

This project is an interactive HR Analytics Dashboard built using Microsoft Power BI. It provides insights into employee engagement, satisfaction, and work-life balance across different departments.

## Dataset

The dataset contains employee information including:

- Employee ID
- Department
- Engagement Score
- Satisfaction Score
- Work-Life Balance Score
- Employee Status

## Dashboard Features

- KPI Cards
  - Total Employees
  - Average Engagement
  - Average Satisfaction
  - Average Work-Life Balance

- Department Filter (Slicer)

- Interactive Bar Charts
  - Average Engagement by Department
  - Average Satisfaction by Department
  - Average Work-Life Balance by Department

## Tools Used

- Microsoft Power BI
- CSV Dataset
- GitHub
